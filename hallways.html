Difference between block/inline elements

  • block-level = stack on top, occupy any width. can nest inside eachother, can wrap inline-level elements. usually bigger pieces of content (like p's)
  • inline-level = don't begin on new line. fall into normal flow of doc, line up one after other, maintain width of their content. can nest inside each other, but can't wrap block-level els. usually smaller content, (like words)

CSS specificity points - trump casc position

  • type selector -- lowest specif weight = 0-0-1
  • class selector -- medium specif weight = 0-1-0
  • id selector -- high specif weight = 1-0-0
